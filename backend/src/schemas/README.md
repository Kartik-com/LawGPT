# Schema Documentation

This directory contains schema documentation for the cloud services used in the Lawyer Zen application.

## Files

- **firebase-schema.md** - Firebase Authentication schema and structure
- **cloudinary-schema.md** - Cloudinary media storage schema and structure

## Purpose

These schema files provide:
- Detailed structure of data stored in each service
- Authentication and storage flows
- Security considerations
- Migration guides
- Best practices

## Quick Reference

### Firebase Auth
- User credentials stored in Firebase
- Custom claims for role-based access
- MongoDB stores additional profile data

### Cloudinary
- All media files stored in Cloudinary
- MongoDB stores file metadata
- Organized by user and folder structure

See individual schema files for detailed information.
